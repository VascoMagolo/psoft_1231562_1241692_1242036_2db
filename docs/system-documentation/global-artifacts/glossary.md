# Glossary

**Terms, Expressions and Acronyms (TEA) — sorted alphabetically by TEA (PT)**

| **_TEA_ (PT)**                                    | **_TEA_ (EN)**                            | **_Description_ (EN)**                                                                                                                                        |
|:--------------------------------------------------|:------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Administrador do Sistema                          | System Administrator                      | A user role responsible for initializing system configuration and managing credentials for administrative users.                                              |
| Aeronave                                          | Aircraft                                  | An entity representing a specific aircraft instance identified by a registration number, associated with a model, manufacturing date, and operational status. |
| Aeroporto                                         | Airport                                   | An entity representing a physical airport identified by an IATA code, including name, location, coordinates, operational status, and runway infrastructure.   |
| Autonomia                                         | Range                                     | The maximum distance an aircraft can travel without refueling; used to determine compatibility with flight routes.                                            |
| Capacidade de Passageiros                         | Seating Capacity                          | The maximum number of passengers an aircraft (or configuration) can carry.                                                                                    |
| Características da Aeronave                       | Aircraft Features                         | Specific attributes, amenities, or technical features associated with an aircraft instance (e.g., WiFi-enabled).                                              |
| Certificação de Aeronave                          | Airplane Certification                    | An official authorization indicating that a specific aircraft model is permitted to operate at a given airport.                                               |
| Código IATA                                       | IATA Code                                 | A unique three-letter code used to identify an airport.                                                                                                       |
| Colaborador da Empresa de Transporte Aéreo (ATCC) | Air Transport Company Collaborator (ATCC) | A user role responsible for managing aircraft, routes, and flight operations within an air transport company.                                                 |
| Componente de Manutenção                          | Maintenance Component                     | A classification category for maintenance activities (e.g., engine, airframe, avionics).                                                                      |
| Contacto do Aeroporto                             | Airport Contact                           | A structured communication method (e.g., phone, email, fax) associated with a specific airport or its departments.                                            |
| Disponibilidade                                   | Availability                              | A derived state indicating whether an aircraft is eligible for assignment to flights, based on its operational status.                                        |
| Estado da Aeronave                                | Aircraft Status                           | The current operational condition of an aircraft (e.g., Active, Inactive, Under Maintenance).                                                                 |
| Estado da Rota                                    | Route Status                              | The current operational condition of a flight route (e.g., Active, Deactivated).                                                                              |
| Estado do Aeroporto                               | Airport Status                            | The operational condition of an airport (e.g., Operational, Closed, Under Maintenance).                                                                       |
| Estado do Voo                                     | Flight Status                             | The current lifecycle state of a scheduled flight (e.g., Scheduled, Delayed, In-Flight, Completed, Canceled).                                                 |
| Fabricante de Aeronave                            | Aircraft Manufacturer                     | The company responsible for producing an aircraft model.                                                                                                      |
| Instalações do Aeroporto                          | Airport Facilities                        | Detailed infrastructure elements available at an airport, specifically encompassing Terminals, Gates, and Services.                                           |
| Intervalo de Manutenção                           | Maintenance Interval                      | The defined threshold, measured in accumulated flight hours or calendar days, that triggers a scheduled maintenance alert for an aircraft.                    |
| Lista de Verificação                              | Checklist                                 | A structured list of tasks defined within a maintenance template to be executed during a maintenance event.                                                   |
| Modelo de Aeronave                                | Aircraft Model                            | A specification describing a class of aircraft, including manufacturer, model name, fuel capacity, range, and seating capacity.                               |
| Modelo de Manutenção                              | Maintenance Template                      | A reusable definition of a maintenance procedure, including type, applicable aircraft models, and associated checklist.                                       |
| Operador de Backoffice                            | Backoffice Operator                       | A user role responsible for managing reference data such as aircraft models and airports.                                                                     |
| Peça de Manutenção                                | Maintenance Part                          | An inventory item used in maintenance operations, tracked by stock quantity and a minimum threshold for automated low-stock alerts.                           |
| Pista                                             | Runway                                    | An airport infrastructure element defined by its identifier, length, and orientation, used for aircraft takeoff and landing.                                  |
| Região                                            | Region                                    | A broad geographical area (e.g., Europe, North America) used to group and filter airports beyond the country level.                                           |
| Registo de Manutenção                             | Maintenance Record                        | An entity representing a specific maintenance event performed on an aircraft, including duration, type, and executed tasks.                                   |
| Rota de Voo                                       | Flight Route                              | An entity representing a predefined path between an origin and a destination airport, with constraints such as minimum range and capacity requirements.       |
| Supervisor de Manutenção                          | Maintenance Supervisor                    | A user role responsible for monitoring maintenance operations and analyzing performance metrics such as turnaround time.                                      |
| Técnico de Manutenção                             | Maintenance Technician                    | A user role responsible for creating maintenance templates and executing maintenance records.                                                                 |
| Tipo de Manutenção                                | Maintenance Type                          | A classification of maintenance activities (e.g., Inspection, Scheduled Maintenance, Overhaul, Modification).                                                 |
| Voo Programado                                    | Scheduled Flight                          | An entity representing a scheduled assignment of an aircraft to a flight route at a specific date and time.                                                   |
